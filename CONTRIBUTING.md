# Contributing to File Storage Adapter

Thank you for considering contributing to this project! We welcome contributions of all kinds, including bug fixes, documentation improvements, or new features.

## How to Contribute

1. **Fork the Repository**
    - Go to the main repository page and click on the "Fork" button. Clone your fork to your local machine.
    - Example: `git clone https://github.com/YOUR-USERNAME/file-storage-adapter.git`

2. **Work on an Issue (Optional)**
    - Check out the existing issues for areas where contributions are needed or suggest a new change.
    - If you work on an issue, comment on it so everyone knows it is being worked on.

3. **Create a New Branch**
    - Create a branch for your work so like: `git checkout -b your-branch-name`
    
4.  **Make Your Changes and Commit**
